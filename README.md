# Firebase Android Series 🚀

This repository contains a complete Firebase Android Development series using Kotlin and Android Studio.

It is designed for beginners to learn Firebase step by step with real-world implementation.

---

## 📌 About Project
This project covers Firebase integration from basics to advanced concepts including Authentication, Realtime Database, CRUD operations, and Security Rules.

---

## 🚀 Features

- Firebase Setup in Android Studio
- Email & Password Authentication
- Google Sign-In Authentication
- CRUD Operations (Create, Read, Update, Delete)
- Firebase Realtime Database
- Realtime Listeners (ValueEventListener, ChildEventListener)
- Firebase Security Rules
- Real-time data updates in UI
- Clean and beginner-friendly code

---

## 🛠 Tech Stack

- Kotlin  
- Android Studio  
- Firebase Realtime Database  
- Firebase Authentication  
- XML Layouts  

---

## 📂 Project Structure

- Part 1 - Firebase Setup  
- Part 2 - Authentication  
- Part 3 - CRUD Operations  
- Part 4 - Realtime Database  

---

## ⚠️ Important Note

This repository does NOT include `google-services.json` file for security reasons.  

To run this project:
1. Create your own Firebase project
2. Add Android app in Firebase Console
3. Download `google-services.json`
4. Place it inside the `app/` folder
5. Sync Gradle and run the project
